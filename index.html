FunPassGen

by Alex Diwa and Jade Rosario

FunPassGen - or Fun Password Generator - is a terminal application written in Ruby. FunPassGen generates a secure password based on the user's responses to prompts, as well as a memorable story that enables easy password recall. Additionally, the app can encrypt the password for the user.
